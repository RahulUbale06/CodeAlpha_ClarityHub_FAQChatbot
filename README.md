# ⚡ Clarity Hub - AI FAQ Assistant

An AI-powered FAQ Assistant built using **Python**, **Streamlit**, **TF-IDF Vectorization**, and **Cosine Similarity** to provide intelligent semantic search and instant knowledge retrieval.

---

## 📌 Project Overview

Clarity Hub is a smart FAQ Assistant that allows users to search a knowledge base using natural language queries. Instead of relying on exact keyword matching, the system uses semantic similarity techniques to identify the most relevant answer.

The application is designed with a modern enterprise-style interface and provides intelligent documentation retrieval with confidence scoring and explainable search results.

---

## 🚀 Features

### 🔍 Semantic Search
- Understands user intent beyond exact keyword matches.
- Retrieves the most relevant answer using NLP techniques.

### 🧠 TF-IDF + Cosine Similarity
- Converts questions into numerical vectors.
- Calculates similarity scores to rank answers.

### 📊 Knowledge Analytics
- Displays total knowledge entries.
- Shows available domains.
- Tracks helpful responses.

### 🎯 Confidence Scoring
- Shows match strength for each query.
- Helps users understand result accuracy.

### 📁 Domain Filtering
- Filter the knowledge base by category.
- Quickly narrow down search results.

### 🧩 Typo Recovery
- Detects and handles minor spelling mistakes.
- Improves user experience and search accuracy.

### 💡 Explain Match Logic
- Displays how the system generated the result.
- Provides transparency using NLP metrics.

### 📥 Export Search Session
- Download search history as JSON.
- Useful for documentation and reporting.

### 🎨 Modern UI
- Nordic Midnight Theme
- Enterprise dashboard layout
- Interactive cards and analytics panel

## 🧠 How It Works

1. User enters a query.
2. The query is transformed into a TF-IDF vector.
3. All knowledge base questions are vectorized.
4. Cosine Similarity calculates similarity scores.
5. The highest-ranking answer is returned.
6. Confidence score and match explanation are displayed.
